
# Python Weather App 🌦️

This is a simple **Weather Application** built using **Python Flask** and the **OpenWeatherMap API**.  
The app allows users to enter a city name and view real-time weather information.

## Features
- Search weather by **city name**
- Displays **current temperature**
- Shows **minimum and maximum temperature**
- Displays **humidity and wind speed**
- Shows **weather description and icon**
- Clean and simple interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Run the Project

1. Clone the repository 
2. Go to the project folder
3. Install required libraries
4. Add your **OpenWeatherMap API key** inside `app.py`
5. Run the application
6. Open the **index.html** file in your browser.

## How to Use
1. Enter a **city name**
2. Click **Get Weather**
3. The app will display:
   - Temperature
   - Min / Max Temperature
   - Weather Description
   - Humidity
   - Wind Speed
